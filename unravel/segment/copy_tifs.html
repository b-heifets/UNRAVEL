unravel.segment.copy_tifs module#

Use seg_copy_tifs from UNRAVEL to copy a subset of .tif files to a target dir for training ilastik.

Usage to prep for seg_brain_mask:#

seg_copy_tifs -i reg_inputs/autofl_50um_tifs -s 0000 0005 0050 -o brain_mask

Usage to prep for seg_ilastik to segment full resolution immunofluorescence images:#

seg_copy_tifs -i raw_tif_dir -s 0100 0500 1000 -o ilastik_segmentation

unravel.segment.copy_tifs.copy_specific_slices(sample_path, source_dir, target_dir, slice_numbers, verbose=False)[source]#

Copy the specified slices to the target directory.

Parameters:

  • sample_path (-) – Path to the sample directory.

  • source_dir (-) – Path to the source directory containing the .tif files.

  • target_dir (-) – Path to the target directory where the selected slices will be copied.

  • slice_numbers (-) – List of slice numbers to copy.
